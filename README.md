üìä Dataset Contents
The dataset captures historical car sales data with the following key attributes:

Column Name	Description
Date	Transaction date (daily granularity)
Transaction_ID	Unique identifier for each sale
Dealer_ID	Identifier for the dealership where the sale occurred
Car_Model	Model name of the vehicle sold
Car_Make	Manufacturer/brand of the vehicle
Year	Manufacturing year of the vehicle
Price	Sale price of the vehicle
Discount	Discount applied (if any)
Customer_Segment	Segment classification (e.g., individual, corporate, fleet)
Region	Geographic region of the dealership
Inventory_Status	Inventory flag (e.g., in-stock, sold, backorder)
Salesperson_ID	Identifier for the salesperson handling the transaction
